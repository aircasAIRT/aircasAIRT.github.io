AIRT:创建一篇新博客

发表于 2022-09-19 分类于 说明文档

本文主要讲述如何在服务器端发布文章

发布文章的操作主要需要两点操作：

  • 本地连接服务器
  • markdown编辑

由于VScode的便捷可行，本文主要介绍基于 VScode 的博客发文流程，（也可通过其他软件完成以上两点需求）。相关方案以及参考如下：

需求 方案 参考
ssh连接服务器 VScode + Remote-SSH Remote-SSH使用
markdown编辑 VScode + Markdown Preview Enhanced Markdown基本语法

准备工作

首先需要在 VScode 中安装插件：在 VScode 扩展里搜索 Remote-SSH 和 Markdown Preview Enhanced 并安装。

免密登陆设置

由于每次登录或者更换路径都需要输入服务器密码，因此可以对本地免密登录服务器进行设置，具体流程为：

本地电脑打开 cmd，依次输入：

cd ~/.ssh

ssh-keygen -t rsa

ssh-copy-id -i ~/.ssh/id_rsa.pub user_name@IP
执行后按照提示输入密码。

ssh user_name@IP 测试是否免密设置成功。
